Bitcoin ECDSA Calculator

A standalone Bitcoin elliptic curve calculator that works entirely in your browser

Offline Calculator

This calculator allows you to perform various ECDSA cryptographic operations for Bitcoin:

  • Crack a private key using two signatures that reused the same nonce (k)
  • Crack a private key using a known nonce (k)
  • Calculate a public key from a private key
  • Sign a message with a secret key
  • Verify a message signature
  • Find a public key from a signature
  • Find the signing secret (k) used for a signature

You can use the calculator in two ways:

Download Standalone HTML Calculator Open Calculator In New Tab
The standalone calculator works completely in your browser with no server connections needed. All cryptographic operations are performed locally using JavaScript.
